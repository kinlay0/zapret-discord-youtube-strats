# zapret-discord-youtube-strats

Все стратегии собраны в одном месте, чтобы вы могли протестировать их буквально в несколько нажатий.

legacy - старые версии конфигов (например, `LEGACY4` - это стратегия из предыдущей версии `ALT4`). они оставлены как резервные на случай, если новые стратегии на вашем провайдере не сработают.

Огромная благодарность [bolvan](https://github.com/bol-van/zapret), [flowseal](https://github.com/Flowseal/zapret-discord-youtube), [v3nilla](https://github.com/V3nilla/IPSets-For-Bypass-in-Russia) и многим другим, кто создал различные стратегии.
